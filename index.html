Provisioning

This section allows provisioning of existing entities such as users, groups and filters. Provisioning is a process of assigning roles - within any particular application - to an entity. After selecting an entity to provision there are three ways to choose applications you'd like to work with. First is to search for an application using a type-ahead. Second is to go through an alphabetically sorted list. Third is to apply another entity as a template.

Start Provisioning »

Creation

This section allows the creation of new entities: Groups and Filters. A group is a collection of users chosen from a type-ahead, list or based on another group as a template. A filter is a sort of a condition and as such becomes a type of a dynamic group. A filter is created by choosing all or some appropriate criteria to dynamically select users by.
